# C++ Raytracer

---

## Overview

A modular, Whitted-style raytracer written in **C++17** from scratch.  
This project implements:

- Recursive ray tracing  
- BVH acceleration  
- Texture mapping  
- Distributed effects:
  - Soft shadows  
  - Glossy reflections  
  - Motion blur  
  - Depth of field  

---

## Dependencies

- **Compiler:** g++ with C++17 support  
- **Build System:** GNU Make  
- **Libraries:** C++ Standard Library

---
